# Generador-de-Quizzes-Automatico

## Cómo usar la aplicación:

### Instalación:
1. **Copia el código** del artifact a un archivo llamado `index.html`
2. **Abre VSCode** y abre ese archivo
3. **Instala Live Server** (si no lo tienes): Ve a Extensiones y busca "Live Server" de Ritwick Dey
4. **Click derecho** en el archivo → "Open with Live Server"

### Uso de la aplicación:

**Formato de las preguntas:**
```
P: ¿Cuál es la capital de España?
A) Barcelona
B) Madrid *
C) Valencia
D) Sevilla

P: ¿Cuántos planetas hay en el sistema solar?
A) 7
B) 8 *
C) 9
D) 10
```

##  Características implementadas:

- **Aleatorización completa**: Las preguntas y las opciones se mezclan en cada intento  
- **Feedback inmediato**: Muestra si acertaste o fallaste al instante  
- **Interfaz moderna**: Diseño limpio con gradientes y animaciones  
- **Barra de progreso**: Ves cuántas preguntas llevas respondidas  
- **Pantalla de resultados**: Estadísticas completas al finalizar  
- **100% local**: Sin conexión a internet, sin backend, sin bases de datos  
- **Responsive**: Funciona en cualquier tamaño de pantalla  

## Funcionalidades:

- **Parseo inteligente**: Detecta automáticamente preguntas, opciones y respuestas correctas
- **Validación**: Te avisa si el formato no es correcto
- **Repetir quiz**: Puedes volver a hacer el mismo quiz con preguntas mezcladas
- **Nuevo quiz**: Crea un quiz completamente diferente
